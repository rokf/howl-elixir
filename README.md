**Includes**:
- dynamic **completion** and **docs** with respect for aliases (via `elixir` executable)
- code inspection and formating with **mix credo** and **mix format**
- syntax highlighting

**Installation:**
- `git clone` into `~/.howl/bundles`

**Dependencies:**
- `lpeg` (@**luarocks**)
- `serpent` (@**luarocks**)
- `lfs` (@**luarocks**)
- `elixir` and `mix` ofc
